HTTP version of AWE.write will shutdown soon

There are two version of AWE.write:

  • http://awewrite.de
  • https://awewrite.de

After May 31st the http version will no longer be available.

You need to save your data before that happens. Otherwise you won't be able to access your texts from the http version any longer.

To do that:

  • Go to https://awewrite.de/app and copy 'n' paste your texts from the http version to the https one.
  • Update your bookmarks, if any.

Then you're safe.

As soon as the http version is taken offline it will automatically redirect to the https version.
